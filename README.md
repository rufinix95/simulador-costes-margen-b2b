# 📊 Simulador Financiero de Márgenes y Calculadora de Costes de Producción B2B

## 📝 Visión General del Proyecto
Este software de escritorio (.exe) fue desarrollado para digitalizar y agilizar radicalmente el proceso de tarificación y presupuestación de materiales especiales (vidrios complejos) y escandallo de componentes en entornos de manufactura industrial B2B.

La herramienta centraliza las tarifas variables de múltiples proveedores y calcula de forma automatizada e instantánea los precios finales de venta aplicando recargos operativos, sobrecostes de producción y márgenes comerciales dinámicos que requiera la dirección.

---

## 🎯 El Problema del Mundo Real (Retrasos Comerciales y Cuellos de Botella)
El flujo de trabajo anterior para presupuestar materiales no estandarizados dependía totalmente de la disponibilidad manual de la dirección, generando ineficiencias críticas:

1. **Ciclo de Venta Paralizado:** Cuando un comercial o cliente solicitaba un precio ajustado para competir en una licitación, la dirección debía consultar manualmente los tarifarios físicos o Excel aislados de los proveedores. Este proceso podía demorarse horas o incluso días, frustrando al cliente y transmitiendo una imagen de baja profesionalidad.
2. **Pérdida de Oportunidades:** Si la dirección no estaba disponible, la cotización se detenía, provocando que la empresa perdiera obras o quedara fuera de procesos competitivos por no responder a tiempo.
3. **Riesgo Operativo:** Al realizar combinaciones complejas de materiales y recargos operativos de forma manual, aumentaba el riesgo de errores en los cálculos, comprometiendo la rentabilidad neta de la operación.

## 🚀 Solución y Funcionalidades Desarrolladas
El sistema cuenta con una interfaz gráfica intuitiva diseñada para agilizar la toma de decisiones financieras y comerciales:

### 1. Módulo de Gestión de Tarifas Base
* **Funcionalidad:** Centraliza la información de los proveedores y permite la carga ilimitada de materiales y sus precios de coste actualizados.
* **Impacto:** Actúa como una base de datos local unificada que mantiene actualizadas las tarifas de compra de la empresa.

### 2. Simulador y Calculadora de Escandallo Comercial
* **Funcionalidad:** El usuario selecciona la combinación de materiales (ej. tipo de vidrio), añade recargos operativos (ej. *Recargo de Perfil* o *Recargo de Dimensiones*), sobrecostes de producción (ej. *Sueldos operarios sábado*) y aplica un margen de beneficio porcentual dinámico.
* **Impacto:** El software procesa todos los datos y genera el **precio final de venta en segundos**, eliminando la necesidad de consultar manualmente a la dirección.
